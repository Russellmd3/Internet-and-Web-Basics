HTML MCQ Questions

  1. What does HTML stand for?
    1. HyperText Markup Language
    2. Hyper Transfer Markup Language
    3. Hyperlinks and Text Markup Language
    4. Hyper Transfer Model Language

    Answer: HyperText Markup Language

  2. What is the purpose of the <head> tag in HTML?
    1. To define the header of the webpage
    2. To contain metadata like title, character set, and link to stylesheets
    3. To display the main content of the webpage
    4. To define the body section of the webpage

    Answer: To contain metadata like title, character set, and link to stylesheets

  3. What is an HTML editor used for?
    1. To run HTML code
    2. To design HTML pages
    3. To edit and write HTML code
    4. To debug HTML code

    Answer: To edit and write HTML code

  4. Which of the following is the correct HTML element for inserting a line break?
    1. <br>
    2. <break>
    3. <lb>
    4. <hr>

    Answer: <br>

  5. Which tag is used to create a hyperlink in HTML?
    1. <link>
    2. <a>
    3. <href>
    4. <url>

    Answer: <a>

  6. Which tag is used to define a table in HTML?
    1. <table>
    2. <td>
    3. <tr>
    4. <th>

    Answer: <table>

  7. Which tag is used for making text italic in HTML?
    1. <i>
    2. <em>
    3. Both A and B
    4. <italic>

    Answer: Both A and B

  8. What does the align attribute do in the <img> tag in HTML?
    1. Aligns the image to the center
    2. Aligns the image to the left or right
    3. Defines the width of the image
    4. Changes the imageâ€™s format

    Answer: Aligns the image to the left or right

  9. How do you specify an external CSS file for a webpage in HTML?
    1. <style src="style.css">
    2. <css href="style.css">
    3. <link rel="stylesheet" href="style.css">
    4. <script src="style.css">

    Answer: <link rel="stylesheet" href="style.css">

  10. Which tag is used to create in HTML?
    1. <dt>
    2. <dd>
    3. <dl>
    4. <dtlist>

    Answer: Both A and B

  11. Which HTML element is used to define a division or section in an HTML document?
    1. <section>
    2. <div>
    3. <article>
    4. <group>

    Answer: Both A and B

  12. What is the correct HTML tag to use for a video?
    1. <media>
    2. <movie>
    3. <video>
    4. <film>

    Answer: <video>

  13. How can you make an ordered list with Roman numerals in HTML?
    1. <ol type="I">
    2. <ol type="roman">
    3. <ol type="l">
    4. <ol style="list-style-type: roman;">

    Answer: <ol style="list-style-type: roman;">

  14. How do you create a placeholder text in an HTML input field?
    1. placeholder="Enter text"
    2. input="Enter text"
    3. value="Enter text"
    4. hint="Enter text"

    Answer: Both A and D

  15. Which tag is used to define a table header cell in HTML?
    1. <th>
    2. <td>
    3. <tr>
    4. <thread>

    Answer: <th>
